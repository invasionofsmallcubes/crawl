Dungeon Crawl Stone Soup

Tiles · Graphics
Tiles
Sprite-based graphics via SDL2 and WebGL. Dungeon, HUD, and menus rendered with tiles.
Optimised for iOS mobile: extra title splashes culled and tile atlases pngquant'd so the bundle fits inside Safari's per-tab WebAssembly memory ceiling. Without these changes the game crashed.
